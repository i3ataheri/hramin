# حرمین شریفین - گالری عکس

گالری عکس تعاملی حرمین شریفین (مکه و مدینه) با پشتیبانی از ۲۰ زبان مختلف.

## ویژگی‌ها

- طراحی واکنش‌گرا با رابط کاربری شبیه تلفن همراه
- پشتیبانی از ۲۰ زبان مختلف
- نقشه تعاملی با تمام ۲۲ مکان مقدس
- گالری عکس با اسلایدر افقی
- توضیحات کامل هر مکان به زبان‌های مختلف

## نحوه اجرا

```bash
npm install
npm run dev
```

## نحوه ساخت

```bash
npm run build
```

## ساختار پروژه

```
src/
├── pages/
│   ├── index.astro      # صفحه اصلی
│   └── map.astro        # نقشه تعاملی
├── layouts/
│   └── App.astro        # لایوت اصلی
├── styles/
│   └── global.css       # استایل‌ها
├── data/
│   ├── places.json      # اطلاعات مکان‌ها
│   └── lang/            # فایل‌های ترجمه
│       ├── ar.json
│       ├── fa.json
│       ├── en.json
│       └── ... (۲۰ زبان)
public/
├── images/              # تصاویر مکان‌ها
│   ├── makkah/
│   │   ├── haram/
│   │   ├── hira/
│   │   └── ...
│   └── madinah/
│       ├── nabawi/
│       ├── baqi/
│       └── ...
├── fonts/               # فونت‌ها
└── lang/                # فایل‌های ترجمه (public)
```

## فونت‌ها

- Inter: زبان‌های لاتین و سیریلیک
- Vazirmatn: فارسی و پشتو
- Amiri: عربی
- Gulzar: اردو
- Noto Sans Bengali: بنگالی
- Noto Sans SC: چینی

## نقشه

نقشه تعاملی با استفاده از Leaflet.js و کاشی‌های CartoDB Dark طراحی شده است. تمام ۲۲ مکان مقدس روی نقشه نمایش داده می‌شوند.
