# حرمین شریفین - گالری عکس

گالری عکس تعاملی حرمین شریفین (مکه و مدینه) با پشتیبانی از ۲۰ زبان مختلف.

## ویژگی‌ها

- طراحی واکنش‌گرا با رابط کاربری شبیه تلفن همراه
- پشتیبانی از ۲۰ زبان مختلف
- گالری عکس استوری‌مانند با اسکرول افقی
- توضیحات کامل هر مکان به زبان‌های مختلف
- صفحه فروشگاه با اطلاعات محصولات و فروشندگان

## نحوه اجرا

```bash
npm install
npm run dev
```

## نحوه ساخت

```bash
npm run build
```

## ساختار پروژه

```
src/
├── pages/
│   ├── index.astro        # صفحه انتخاب زبان
│   ├── welcome.astro      # صفحه خوش‌آمدگویی
│   ├── gallery.astro      # گالری عکس استوری
│   └── shop.astro         # صفحه فروشگاه
├── layouts/
│   └── App.astro          # لایوت اصلی
├── styles/
│   ├── language.scss      # استایل صفحه زبان
│   ├── welcome.scss       # استایل صفحه خوش‌آمدگویی
│   ├── gallery.scss       # استایل گالری
│   ├── place-menu.scss    # استایل منوی مکان‌ها
│   └── shop.scss          # استایل فروشگاه
public/
├── data/
│   └── places.json        # اطلاعات ۲۲ مکان
├── lang/                  # فایل‌های ترجمه (۲۰ زبان)
├── images/                # تصاویر مکان‌ها
│   ├── makkah/
│   └── madinah/
├── fonts/                 # فونت‌ها
├── flags/                 # پرچم‌ها
├── icons/                 # آیکون‌ها
├── app.js                 # منطق گالری
└── place-menu.js          # منوی مکان‌ها
```

## فونت‌ها

- Amiri: عربی
- Vazirmatn: فارسی، پشتو
- Gulzar: اردو
- Inter: سایر زبان‌های لاتین و سیریلیک
- Noto Sans Bengali: بنگالی
- Noto Sans SC: چینی
