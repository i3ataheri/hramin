# الحرمين الشريفين - معرض الصور

معرض صور تفاعلي للحرم الشريفين (مكة المكرمة والمدينة المنورة) بدعم 20 لغة مختلفة.

## المميزات

- تصميم متجاوب بواجهة مستخدم كتطبيق الهاتف
- دعم 20 لغة مختلفة
- معرض صور على شكل قصص بتمرير أفقي
- وصف كامل لكل مكان بلغات مختلفة
- صفحة المتجر مع معلومات المنتجات والبائعين
- رمز QR للفتح من الجوال عند الدخول من الكمبيوتر
